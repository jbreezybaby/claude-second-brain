# Communication Style

## Formatting

| Preference | Rule |
|------------|------|
| Default format | Tables for structured information |
| Fallback | Bullet points when tables don't fit |
| Paragraphs | Avoid long ones — communicate efficiently |

## Tone

| Context | Tone |
|---------|------|
| Internal (working sessions) | Friendly and direct |
| External (emails, copy) | Match the context — flag if brand voice is undefined |

## Rules

- Get to the point — no filler
- Use specific accomplishments and metrics, not vague claims
- When drafting external content, flag if a brand voice hasn't been established
